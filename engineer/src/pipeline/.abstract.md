Pipeline engine: 17 Python modules, ~6100 LOC. Core chain: models -> loader -> validator -> state -> slot_registry -> gate_checker -> runner. Plus: auto_executor, bootstrap, nl_matcher, observer, slot_contract, enforcer, ov_context_router, project_planner, pipeline_generator, __init__. AutoExecutor automates the full execution loop on top of PipelineRunner (agent resolution + contract generation + parallel execution + output validation + retry). Bootstrap module provides one-call `boot()` entry point that initializes all components with sensible defaults + `BootstrappedExecutor` wrapper with `run_nl()` for natural-language-triggered execution. Meta-orchestration via project_planner.py (blueprint data model) + pipeline_generator.py (generates pipelines from blueprints). OpenViking integration via ov_context_router.py (subprocess CLI calls, no Python imports).
