# architect/ -- Architect Workspace

Design documents, delivery artifacts, and evaluation reports produced by ARCH-001.

## Files

### architecture.md (v1.0.0, ~41KB)
The master system architecture document. Key chapters:
1. **System Vision** -- Pluggable, Lego-like universal pipeline engine. Core insight: topology and personnel are orthogonal. Unix pipes mental model.
2. **Core Abstractions** -- Pipeline, Slot, SlotType, SlotAssignment, DataFlow, Gate, Parameter, ExecutionConfig. Concept map showing relationships.
3. **Design Principles** -- Interface-first, validate early/fail fast, explicit over implicit, minimum viable schema. Priority: Correctness > Simplicity > Extensibility > Performance > Elegance.
4. **Module Decomposition** -- 9-module architecture (models -> loader -> validator -> state -> slot_registry -> gate_checker -> runner -> nl_matcher -> __init__).
5. **Extension Points** -- How to add new slot types, templates, condition checkers, observers.
6. **Architecture Decision Records** -- Key decisions with rationale.

### DELIVERY.yaml
Delivery manifest for WP-2.2 (template conversion to slot-based v2.0.0) and WP-2.3 (schema + implementation guide update). 9 files delivered: 5 converted templates + 2 new slot types (deployer, auditor) + schema v2.0 + implementation guide v2.0. All verification steps passed.
